# 📊 Smart Resume Analyzer (ATS Checker)

A web-based application that analyzes resumes against job descriptions and calculates an **ATS (Applicant Tracking System) compatibility score**. The system compares keywords in the resume with keywords in the job description and highlights missing skills that may improve the chances of getting shortlisted.

---

## 📌 Project Overview

Recruiters often use **Applicant Tracking Systems (ATS)** to filter resumes before human review. This project simulates a basic ATS system by comparing resume content with job descriptions and generating a compatibility score.

The application helps users:

- Understand how well their resume matches a job description  
- Identify missing keywords or skills  
- Improve their resume for better job selection chances  

---

## 🚀 Features

- Resume and job description comparison  
- Automatic keyword extraction and matching  
- ATS compatibility score calculation  
- Highlighted missing keywords
- Color-coded score display (Good / Medium / Poor)  
- Modern UI with gradient background and glassmorphism design  

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ How It Works

1. The user pastes the **resume content** into the resume input box.
2. The user pastes the **job description** into the job description input box.
3. The system extracts keywords from the job description.
4. It compares those keywords with the resume text.
5. The application calculates an **ATS score** and displays missing keywords.
